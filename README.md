# PolyCrushLod

## Installing

Download the repository

```bash
git clone git@github.com:nosleepfilipe/PolyCrushLod.git
```

##Then install the npm modules:

````
cd PolyCrushLod 
git checkout master 
npm install
````

Now you should be ready to develop!


## To start the server

```
node server.js
```
